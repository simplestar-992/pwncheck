# PWNCHECK

**Check if Passwords are Pwned**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/pwncheck.git
cd pwncheck
go build -o pwncheck .
```

## Usage

```bash
./pwncheck --help
```
